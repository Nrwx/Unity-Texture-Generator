# backend

## Project setup
```
cd backend
pip install -r requirements.txt
```

### venv
```
# Linux/macOS
source venv/bin/activate

# Windows
venv\Scripts\activate
```

### Anaconda files
```
conda install -c conda-forge cairo pango gdk-pixbuf libxml2 libffi
```

### Start App
```
cd backend
python app.py
```
